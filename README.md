## gesture 4 book

一个针对博阅阅读器设计的手势工具，在博阅T80D上通过测试。

手机用户要是不嫌界面丑的可以来凑热闹（我不保证不闪退）

### 特性
- 无动画 尽量减少刷新 更适合阅读器
- 操作简单 支持大部分操作
- 免费轻量开源无广告
